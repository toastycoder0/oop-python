# Programación Orientada a Objetos con Python

Ejercicios del curso de Programación Orientada a Objetos con Python del Instituto Tecnológico de Minatitlán.

![Tecnológico de Minatitlán](./tecnm.png)

## Como instalar

Clonar el repositorio

```bash
git clone https://github.com/toastycoder0/poo-python.git
```

Entrar al directorio

```bash
cd poo-python
```
